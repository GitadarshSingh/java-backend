# 🚀 java-backend

> A modular Java backend repository showcasing hands-on development from JDBC and Servlets to Spring Boot, Spring Data JPA, and enterprise-ready backend architecture.

---

## 📌 Overview

This repository represents a structured backend development journey through multiple real Java projects. It demonstrates how backend systems evolve from basic Java programming and database connectivity to scalable REST APIs using the Spring ecosystem.

The projects inside this repository are built for learning, practice, and real implementation experience.

## 📁 Repository Contains

### 🔹 Core Java Projects
Practical Java programs focused on logic building and object-oriented design.

### 🔹 JDBC Projects
Applications performing CRUD operations with MySQL using raw SQL queries.

### 🔹 Servlet Projects
Dynamic web applications using Servlets, request handling, sessions, and deployment structure.

### 🔹 Spring Core Projects
Examples covering dependency injection, bean lifecycle, loose coupling, and XML/annotation configuration.

### 🔹 Spring Boot Projects
Modern standalone backend applications with auto-configuration and REST controllers.

### 🔹 JPA Projects
Entity mapping, repositories, relationships, and database abstraction.

### 🔹 Security Projects
Authentication, authorization, and secure backend development using Spring Security.

### 🔹 Testing Projects
Unit testing service and controller layers using JUnit and Mockito.
